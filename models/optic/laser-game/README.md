# 🛰️ LASER OPS: Security Bypass

A high-tech, interactive physics puzzle game built with **React**, **Vite**, and **Tailwind CSS**. Challenge your understanding of light reflection by manipulating mirrors to bypass security sensors.

## 🚀 Features

- **Real-time Laser Physics:** Dynamic ray-casting and reflection logic using HTML5 Canvas.
- **Interactive Components:** Drag and rotate mirrors with precision controls.
- **High-Tech UI:** A sleek, "Cyberpunk" aesthetic powered by Tailwind CSS and Framer Motion.
- **Telemetry Overlay:** Real-time data visualization showing incidence angles and laser vectors.
- **Sandbox Mode:** Experiment with the physics engine without objective constraints.

## 🎮 How to Play

1.  **Identify the Target:** Locate the hidden override sensor (dotted circle) on the grid.
2.  **Move Mirrors:** Click and drag mirrors **M1** or **M2** to reposition them.
3.  **Rotate:** Hold **Shift** while dragging a mirror to change its angle. Mirrors snap to **15°** increments.
4.  **Bypass:** Reflect the laser beam off the mirrors so it hits the center of the override sensor.

## 🛠️ Tech Stack

- **Framework:** React.js
- **Bundler:** Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React

## 📦 Getting Started

1.  **Clone/Download** the project.
2.  Open your terminal in the project folder and run:
    ```bash
    npm install
    ```
3.  Launch the development server:
    ```bash
    npm run dev
    ```
